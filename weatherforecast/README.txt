To execute the file it's necessary to have installed in your machine:
	- Gradle 4.8.1 or above
	- Java 1.8.0.121 or above

After the project is downloaded execute through prompt
	- gradle clean build (so the jar is generated)
	- gradle bootrun

After application is up you can access through any browser by the URL: http://localhost:8080/data/CITY, where you can replace CITY by any valid city.
For Example: "http://localhost:8080/data/Porto Alegre" or "http://localhost:8080/Berlin"
